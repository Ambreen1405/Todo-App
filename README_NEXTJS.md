# Next.js Todo App

A beautiful and functional todo application built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Add new tasks with title and description
- Mark tasks as complete/incomplete
- Edit existing tasks
- Delete tasks
- View task statistics
- Local storage persistence
- Responsive design
- Modern UI with Tailwind CSS

## Requirements

- Node.js 18 or higher
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- UUID for unique IDs

## Functionality

- **Add Task**: Add new tasks with title and optional description
- **Edit Task**: Click the edit icon to modify existing tasks
- **Complete Task**: Check the checkbox to mark a task as completed
- **Delete Task**: Click the delete icon to remove a task
- **Statistics**: View total tasks, completed tasks, and pending tasks
- **Persistence**: Tasks are saved to localStorage and persist between sessions
